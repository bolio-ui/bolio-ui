## Example with Gatsby

<Spacer h={2} />

### Get started with a example

To begin, the most straightforward approach is to utilize one of the provided templates.
These templates come pre-configured with the necessary settings, such as CssBaseline and other fundamental components.

If you're new to GitHub and need guidance on initiating a project using a template,
you can access comprehensive instructions on how to get started on <Link target="_blank" rel="nofollow" color icon href="https://github.com/bolio-ui/bolio-ui/tree/master/examples/create-gatsby">this example</Link>.

<Spacer h={2} />

### Generate new application

Follow <Link target="_blank" rel="nofollow" color icon href="https://www.gatsbyjs.com/docs/quick-start/"> Gatsby quick start</Link> guide to create new Gatsby application:

<Spacer h={3} />

<Tabs initialValue="yarn" hideDivider hideBorder align="flex-start" mt="-46px">
  <Tabs.Item label="Yarn" value="yarn">

### Setup

Add styles imports, BolioUIProvider and CssBaseline to src/pages/index.tsx:

```jsx
import * as React from 'react'
import type { HeadFC, PageProps } from 'gatsby'
import { BolioUIProvider, CssBaseline, Text } from '@bolio-ui/core'

const IndexPage: React.FC<PageProps> = () => {
  return (
    <BolioUIProvider>
      <CssBaseline />
      <Text p>Your app page here</Text>
    </BolioUIProvider>
  )
}

export default IndexPage

export const Head: HeadFC = () => <title>Home</title>
```

Ready to go! Start the development server:

<Snippet text="npm run develop" />
