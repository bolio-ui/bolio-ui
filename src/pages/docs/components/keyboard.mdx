## Keyboard

Display keyboard input that triggers an action.

The `Keyboard` can only display layout styles, if you need to bind the keyboard events, please read subsection [useKeyboard](/docs/hooks/use-keyboard).

<Playground
  scope={{ Keyboard, Grid }}
  code={`
<Grid.Container gap={2}>
  <Grid>
    <Keyboard>C</Keyboard>
  </Grid>
</Grid.Container>
`}
/>

| **option**  | show option key  | `boolean`              | -                             | `false` |
| **ctrl**    | show ctrl key    | `boolean`              | -                             | `false` |
| ...         | native props     | `KeygenHTMLAttributes` | `'keyType', 'className', ...` | -       |
